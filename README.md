# 醫管師題庫系統
一個基於 Flask + PostgreSQL 的題庫與模擬測驗系統，提供題目查找、練習與測驗功能，輔助學習醫管師考試。

---

## 功能
> 開發中部分功能未完成
- 題庫
  - 一般查詢：關鍵字模糊查詢
  - ✕進階查詢：多條件篩選(題目來源、分類...)
  - ✕題目收藏：使用者可以收藏特定題目，利用錯題重練模式加強練習
  - ✕新增題庫：將來源pdf轉換為固定格式  
    目前`rag_knowledge_base\services\preprocess_pdf.py` 和 `rag_knowledge_base\pdftojson.py`檔案在轉換上還有部分瑕疵
- 模擬測驗
  - 標準模式：從所有題庫中隨機選取題目。
    報表尚未完成
  - 弱點加強模式：優先練習錯誤率較高的題目，提升整體表現。
  - 錯題重練模式：僅包含過去答錯或標記收藏的題目，鞏固學習。
- 錯題分析筆記
  - ✕錯題統計：統計各來源正確率。
  - ✕筆記欄位：使用者可紀錄題目筆記。
  - ✕AI助教：AI解釋題目或建議學習策略。
